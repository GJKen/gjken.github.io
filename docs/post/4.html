Cloudflare 域名 DNS 托管 + GitHub Pages 自定义域名

先决条件

假设你已经:

  • 拥有了一个域名.
  • 域名备案已解决.

Cloudflare 添加域

我们进入 https://dash.cloudflare.com

输入自己的域名点击继续, 然后选择底部的 Free 计划.

复制 dns 名称服务器.

然后转到你的域名提供服务商, 添加 dns 记录.

填写 DNS 记录.

回到 Cloudflare 面板, 进入 dns 记录, 按照下图添加 CNAME 和 A 记录.

185.199.108.153

185.199.109.153

185.199.110.153

185.199.111.153

Github Page 设置自定义域名

填写 Save 之后会提示你等待一会.

提示DNS check successful代表完成, 之后就可以通过www+你的域名进行访问了.
